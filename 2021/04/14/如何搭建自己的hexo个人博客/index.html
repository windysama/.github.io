如何搭建自己的hexo个人博客

如何搭建个人博客

首先我是按照B站UP CodeSheep 的视频教程，按部就班的完成搭建，以下附上视频链接，别忘了点赞三联

https://www.bilibili.com/video/BV1Yb411a7ty?t=1187

然后下面是文字版：

一、安装Nodejs和hexo

node -v #查看node版本

npm -v #查看npm版本

npm install -g cnpm –registry=http://registry.npm.taobao.org #安装淘宝的cnpm 管理器

cnpm -v #查看cnpm版本

cnpm install -g hexo-cli #安装hexo框架

二、开始hexo搭建博客

首先在你电脑某个文件下面创建一个文件夹，可以命名XXblog，用来存放blog

  1. 初始化博客

    hexo init 文件夹名称

  2. 新建文章

    hexo new 文章名称

    (或者)

    hexo n 文章名称

  3. 在线预览

    hexo server

    (或者)

    hexo s

    在线预览可以查看你搭建blog是否成功，通过在浏览器中输入http://loclhost:4000/ 可以预览blo

  4. 一键部署

    hexo deploy

    (或者)

    hexo d
